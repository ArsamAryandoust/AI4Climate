## Getting started

All datasets are provided on Hugging Face Hub. To get started, install the
datasets python package.

```bash
pip install datasets
```

Load a dataset with
```Python
from datasets import load_dataset

dataset = load_dataset("ArsamAryandoust/OPFData")
```
